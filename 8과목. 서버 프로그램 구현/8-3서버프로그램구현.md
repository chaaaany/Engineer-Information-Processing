# 8-3. 서버 프로그램 구현
## 서버 프로그램 구현
: 업무프로세스를 기반으로 개발언어, 도구 이용해 서비스 제공에 필요한 기능을 구현

## VO, SQL문, DAO, Service, Controller 순서대로 구현

**1) DTO, VO구현**

: DB에 저장하는 객체 구현

**2) SQL문 구현**

: DB 테이블 정의 및 생성, XML 구현

**3) DAO(데이터 접근객체) 구현**

: SQL을 구현한 XML ID를 호출해 조작 수행

**4) Service 클래스 구현**

: DAO를 호출하는 서비스를 선언. 구현함.

**5) 컨트롤러 클래스 구현**

**6) 입출력 검증(Validation) 로직 구현**
